# ⏱ Life Timer

Шахматный таймер для планирования дня. Одна задача — одни часы.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Структура проекта

```
life_timer/
├── main.py           # точка входа + главное окно
├── models.py         # модели данных (Task, DayPlan, AppSettings)
├── timer.py          # TimerEngine (логика) + NotificationScheduler
├── storage.py        # сохранение/загрузка JSON (~/.life_timer/)
├── requirements.txt
└── ui/
    ├── timer_header.py    # верхняя панель с активным таймером
    ├── task_panel.py      # список задач
    ├── add_task_dialog.py # диалог добавления задачи
    ├── stats_panel.py     # статистика за 7 дней
    └── settings_dialog.py # настройки
```

## Данные

Сохраняются в `~/.life_timer/`:
- `day_YYYY-MM-DD.json` — план на день
- `settings.json` — настройки

## Что реализовано

- [x] Шахматный таймер: активна одна задача в момент времени
- [x] Отдельный счётчик прокрастинации с лимитом
- [x] Три режима перерасхода (настройки):
  - уходить в минус
  - отъедать от прокрастинации
  - пропорционально от оставшихся задач
- [x] Привязка задач ко времени + уведомления
- [x] Сохранение состояния между запусками
- [x] Статистика за 7 дней
- [x] Автосохранение каждые 30 секунд
